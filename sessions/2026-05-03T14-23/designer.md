모든 디자인 요소는 **'가독성(Readability)'**을 최우선 목표로 하며, 화려함이나 트렌디함은 철저히 배제합니다.

### 🎨 1. 컬러 팔레트 (Color Palette)

**목표:** 배경과 텍스트 간의 대비(Contrast)를 최대화하여, 오랜 시간 읽어도 눈이 피로하지 않도록 합니다.

| 용도 (Use Case) | 색상 이름 | Hex Code | CMYK (인쇄 기준) | 비고 (Comment) |
| :--- | :--- | :--- | :--- | :--- |
| **Primary Background** (배경) | 순백색 (Off-White) | `#FCFCF9` | C: 2 / M: 3 / Y: 4 / K: 0 | 순백색보다 미세하게 따뜻한 톤으로 눈부심 방지. |
| **Primary Text** (본문 텍스트) | 딥 블랙 (Deep Black) | `#222222` | C: 70 / M: 60 / Y: 60 / K: 90 | 순수 블랙보다 부드러우면서도 대비는 최대로 유지. |
| **Accent Orange** (CTA/강조) | 신뢰 오렌지 | `#FF7B2E` | C: 0 / M: 50 / Y: 80 / K: 0 | 기존 강조색 유지. *가장 눈에 띄는 액션 포인트*로만 제한적 사용. |
| **Warning/Caution** (경고/주의) | 안정 노랑 | `#FFD700` | C: 0 / M: 10 / Y: 90 / K: 0 | **절대 강렬한 빨강 사용 금지.** 노란색 배경에 딥 블랙 텍스트로 경고의 심각성을 낮추고 시선을 유도. |
| **Secondary Text** (보조/캡션) | 차분 회색 | `#777777` | C: 40 / M: 30 / Y: 30 / K: 50 | 부가 설명, 출처 등 중요도가 낮은 텍스트에 사용. |

### 🔡 2. 타이포그래피 시스템 (Typography System)

**목표:** 시스템에 맞는 가독성 높은 고딕체 계열을 사용하고, 폰트 크기 최소 기준을 준수합니다.

*   **폰트 선택:** **Noto Sans KR (고딕 계열)** 또는 **맑은 고딕** 등 시스템에서 가장 가독성이 높고 친숙한 고딕 계열을 사용합니다. (명조체는 장문에서 피로도가 높을 수 있음.)
*   **줄 간격 (Line Height):** 본문은 **1.6 ~ 1.8**을 기본으로 설정하여, 문장 간 분리감을 높이고 읽기 편하게 합니다.
*   **최소 크기 기준:** 모든 텍스트 요소는 다음 최소 크기를 준수해야 합니다.

| 요소 (Element) | 폰트 크기 (Size) | 폰트 두께 (Weight) | 색상 (Color) | 용도 및 규칙 (Rule) |
| :--- | :--- | :--- | :--- | :--- |
| **H1 (메인 제목)** | 28pt 이상 | Bold | 딥 블랙 | 가장 크고 명확하게. 콘텐츠의 핵심 주제를 한눈에 전달. |
| **H2 (소제목)** | 22pt 이상 | Semi-Bold | 딥 블랙 | 내용을 덩어리로 구분. 제목과 본문 사이에 충분한 여백(Padding) 필수. |
| **Body Text (본문)** | **18pt 이상** | Regular | 딥 블랙 | **최소 크기 기준.** 한 문단당 3~5줄 이내로 짧게 끊어 작성. |
| **CTA Text (행동 유도 문구)** | 18pt 이상 | Bold | 딥 블랙 | 배경색(Orange)과 대비를 최대화하여 강조. |
| **Caption/Source** (캡션/출처) | 14pt ~ 16pt | Regular | 차분 회색 | 본문 내용의 보조 설명이나 출처 표기에만 사용. |
