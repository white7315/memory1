### 1. 🎨 컬러 팔레트 및 사용 규칙 (Color Palette)

| 영역 | 색상명 | Hex Code | 역할 및 사용 지침 |
| :--- | :--- | :--- | :--- |
| **Primary (전문성)** | 딥 네이비 | `#2C3E50` | **헤딩, 주요 제목, 강조 박스 테두리.** 신뢰감과 권위를 부여하는 주 색상. 본문 텍스트 색상으로 사용 불가 (가독성 저하). |
| **Secondary (배경/본문)** | 아이보리 화이트 | `#FCFCF7` | **전체 배경색.** 순백색(`FFFFFF`)보다 눈의 피로도가 낮아 가독성 최우선. |
| **Tertiary (텍스트)** | 차분한 회색 | `#333333` | **본문 텍스트 색상.** 블랙(`000000`)보다 부드럽고, 딥 네이비보다 가독성이 높아 본문에 사용. |
| **Accent 1 (경고)** | Danger Red | `#C0392B` | **[필수] 경고 문구 박스, 위험 신호 표시.** 주의와 경각심을 주어 시선 집중. **(강한 대비로 사용)** |
| **Accent 2 (행동 유도)** | Vivid Orange | `#FF9900` | **[필수] CTA 버튼, 리드 마그넷 강조.** 즉각적인 행동을 유도하는 활력색. (버튼 배경색으로 사용) |
| **Accent 3 (구분)** | 미디엄 블루 | `#A9D0F0` | **박스 배경, 구분선.** 섹션 전환이나 보조 설명을 할 때 사용. |

### 2. 🖋 타이포그래피 및 가독성 원칙 (Typography & Readability)

| 요소 | 추천 폰트 | 크기 (Size) | 굵기 (Weight) | 줄 간격 (Line Height) | 사용 원칙 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **H1 (메인 제목)** | 나눔고딕 Bold | 32pt ~ 36pt | Bold | 1.3 | 페이지 최상단에 배치. 문제 제기 시 강력하게 사용. |
| **H2 (소제목)** | 나눔고딕 SemiBold | 22pt ~ 26pt | SemiBold | 1.5 | 각 소제목 시작 시 사용. 딥 네이비 색상 사용. |
| **H3 (세부 항목)** | 나눔고딕 Medium | 18pt ~ 20pt | Medium | 1.4 | 목록이나 원리 설명 시 사용. |
| **본문 (Body Text)** | 나눔고딕 / Noto Sans KR | **최소 18pt** | Regular | **1.6 ~ 1.8** | 가장 중요. 글자 크기를 크게 하고, 줄 간격을 충분히 확보하여 눈의 피로도를 낮춤. |
| **CTA/강조 텍스트** | 나눔고딕 Bold | 20pt 이상 | Bold | 1.3 | 오렌지/빨강 강조 시 사용. |

**[공통 레이아웃 원칙]**
1.  **여백(Whitespace):** 섹션 간 최소 30px 이상의 수직 여백을 확보하여 답답함을 해소합니다.
2.  **구분선:** 섹션 전환 시에는 옅은 회색(`E0E0E0`)의 수평 구분선(Divider)을 반드시 삽입합니다.
