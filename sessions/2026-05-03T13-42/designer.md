# 🎨 위젯/챗창 디자인 브리프 (Visual System)

## 1. 🖼️ 위젯/진단 카드 디자인 브리프 (Widget/Diagnostic Card)

위젯은 단순한 광고 배너가 아니라, 사용자가 **'스스로 궁금증을 느끼게 만드는 전문적인 진단 결과의 예고편'** 역할을 해야 합니다.

### 1.1. 디자인 목표 및 컨셉
*   **목표:** 낮은 진입 장벽으로 클릭을 유도하되, 클릭 후 블로그에 도달했을 때 '이건 가벼운 장난이 아니다'라는 전문성을 인지하게 함.
*   **컨셉:** **[Mini-Diagnostic Report Card]** - 마치 컨설팅 보고서의 일부처럼 보이게 디자인하여 신뢰도를 높입니다.
*   **적용 원칙:** 텍스트는 최소화하고, 시각적 요소(아이콘, 그래프, 강조색)를 최대화합니다.

### 1.2. 디자인 사양 (Design Specification)
| 항목 | 상세 사양 | HEX 코드 / 폰트 | 비고 |
| :--- | :--- | :--- | :--- |
| **전체 배경색** | 화이트 (깨끗한 보고서 느낌) | `#FFFFFF` | |
| **주요 강조색 (Primary CTA)** | 신뢰와 경각심을 주는 오렌지 계열 | `#FF6B3D` (Orange) | 클릭 유도 버튼, 핵심 수치 하이라이트 |
| **보조 강조색 (Secondary)** | 기술적이고 명료한 스카이 블루 계열 | `#4A90E2` (Sky Blue) | 아이콘, 진단 항목 구분선 |
| **텍스트 색상** | 다크 그레이 (블랙보다 부드럽게) | `#333333` | 본문 텍스트 |
| **폰트 (Headline)** | Noto Sans KR / Pretendard (Bold) | - | 제목과 수치에 힘을 실어줌 |
| **폰트 (Body)** | Noto Sans KR / Pretendard (Regular) | - | 가독성 위주 |

### 1.3. 레이아웃 (Layout Blueprint)
*   **형태:** 직사각형의 카드 형태 (가로 비율 16:9 권장).
*   **구성 요소 (좌 → 우):**
    1.  **아이콘/진단 주제 (좌측 상단):** 주제와 관련된 전문 아이콘 (예: 📊, 💡, 🚀) + 주제명 (예: "AI 비용 효율 진단").
    2.  **진단 핵심 수치 (중앙):** 가장 임팩트 있는 결과 값 또는 질문 (예: "최대 30% 절감 가능"). 이 부분이 가장 크게, 오렌지색으로 강조됩니다.
    3.  **CTA 버튼 (우측 하단):** "무료 진단 리포트 받기" 또는 "나의 점수 확인하기". 이 버튼은 스카이 블루 또는 오렌지색을 배경으로 명확하게 대비시켜 클릭을 유도합니다.

## 2. 🤖 AI 챗봇 인터페이스 디자인 브리프 (Chatbot UI)

챗봇은 단순한 대화창이 아닌, **'전문가와의 비대면 컨설팅 세션'**을 경험하게 하는 인터페이스여야 합니다.

### 2.1. 디자인 목표 및 컨셉
*   **목표:** 사용자가 AI와 대화하는 과정 자체가 '지적이고 신뢰할 수 있는 경험'이 되도록 설계합니다.
*   **컨셉:** **[Minimalist Professional Dashboard]** - 채팅창의 기능성을 최소화하고, '대화의 흐름'과 '결과물'에 집중합니다.
*   **적용 원칙:** 공백(Whitespace)을 적극적으로 활용하여 답답함과 가벼움을 제거합니다.

### 2.2. 디자인 사양 (Design Specification)
| 항목 | 상세 사양 | HEX 코드 / 폰트 | 비고 |
| :--- | :--- | :--- | :--- |
| **전체 배경색** | 라이트 그레이 또는 오프 화이트 | `#F9F9F9` | 눈의 피로도를 낮추는 배경 |
| **AI 메시지 배경색** | 아주 연한 스카이 블루/민트 계열 | `#E6F7FF` | AI가 말하는 공간임을 시각적으로 구분 |
| **사용자 메시지 배경색** | 화이트 (대비 효과) | `#FFFFFF` | |
| **텍스트 색상** | 다크 그레이 | `#333333` | |
| **폰트** | Noto Sans KR (세련되고 모던한 느낌) | - | 전체적으로 일관된 폰트 사용 |

### 2.3. 레이아웃 (Layout Blueprint)
*   **구조:** 챗봇 창은 페이지의 중앙 하단에 고정된 위젯 형태(Fixed Position Widget)로 배치합니다.
*   **대화 흐름:**
    1.  **헤더:** "JAY CORP AI 비즈니스 컨설턴트" (전문가 명시).
    2.  **메시지 영역:** 일반적인 말풍선 형태를 사용하되, 각 메시지 상단에 **'발신자 이름/역할'**을 작게 명시하여 신뢰도를 높입니다.
    3.  **입력창:** 사용자가 질문을 입력
