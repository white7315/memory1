# 💡 초기 런칭용 대표 위젯 콘텐츠 스크립트 3종

## 🚀 위젯 1: AI 도입 준비도 진단 (AI Readiness Diagnosis)
**[Pain Point]** : AI가 대세인데, 우리 회사만 뒤처지는 건 아닐까? (가장 보편적이고 트렌디한 불안감 자극)
**[주요 유입 채널]** : 네이버, 티스토리 (전문 지식 기반 검색 유입)

### 📋 스크립트 초안 (Widget Flow)

1.  **헤드라인 (Hook):** 🔥 **[AI 도입 준비도 체크]** 우리 비즈니스, AI 시대에 몇 점일까요? (단순한 궁금증 자극)
2.  **위젯 작동 방식 (Action):** 사용자에게 3~5개의 **'Yes/No' 또는 'A/B 선택'** 형태의 질문을 던지는 미니 퀴즈 폼.
    *   *예시 질문:* "마케팅 자료 제작에 매번 외부 인력의 검수가 필요합니까? (Yes/No)"
3.  **결과 화면 (Gap):** 질문에 대한 응답을 바탕으로 **'점수'**와 **'간단한 진단 결과'**를 제시.
    *   *예시 결과:* **"진단 점수: 65/100점 (🚨 주의 단계)"**
    *   *메시지:* "현재 점수만으로는 AI 도입의 **'진짜 문제점'**을 파악하기 어렵습니다. 가장 큰 격차(Gap)는 '운영 효율성'에 있습니다."
4.  **CTA (Conversion):**
    *   **버튼:** ➡️ **[AI 전문가에게 맞춤 진단 받기 (무료)]**
    *   **설명:** "AI 챗봇에 접속하여, 귀사에 맞는 구체적인 개선 로드맵을 받아보세요. (단순 점수표가 아닌, 비용 절감액까지 진단)"

## 💰 위젯 2: 비효율 프로세스 비용 절감 시뮬레이터 (Cost-Saving Simulator)
**[Pain Point]** : 직원들이 매일 반복하는 단순 노동에 너무 많은 시간이 낭비되고 있다. (가장 현실적이고 돈과 직결되는 문제 자극)
**[주요 유입 채널]** : 인스타그램 (릴스/카로셀 마지막 장), 블로그 (재무/운영 효율성 관련 검색)

### 📋 스크립트 초안 (Widget Flow)

1.  **헤드라인 (Hook):** ⏱️ **[월간 숨겨진 비용]** 당신의 업무, 매일 몇 시간을 낭비하고 있나요? (시간과 돈을 직접 언급하여 경각심 유발)
2.  **위젯 작동 방식 (Action):** 사용자가 **'시간'과 '인력'**을 입력하는 간단한 계산기 형태.
    *   *예시 입력:* "매일 반복하는 보고서 요약 작업에 투입되는 시간 (분): [120분]", "해당 작업을 수행하는 인력 수: [2명]"
3.  **결과 화면 (Gap):** 입력된 데이터를 바탕으로 **'월간 예상 낭비 시간'**과 **'잠재적 비용'**을 계산하여 보여줌.
    *   *예시 결과:* **"월간 예상 낭비 시간: 480시간 (💸 최소 1,200만원 이상의 비용 낭비 가능)"**
    *   *메시지:* "이 비용을 줄일 수 있는 방법은 많지만, 어떤 프로세스를 먼저 건드려야 할지 구체적인 우선순위가 필요합니다."
4.  **CTA (Conversion):**
    *   **버튼:** 🤖 **[최적화 프로세스 진단받기]**
